Getting Started

Members - Used to Search, Manage buyer actions, create, modify , delete, reserve and select plots, pay registration amount and view payments.
= Workflow - Add member -> Pay Reg Amt -> Select/Reserve Plot > Pay EMI or Part Payment
= Search the members using name, phone, city and choose the action on the right side to perform.
= Edit link is used to edit the member details , except mobile no
= Del link is used to delete the member from the database
= RAmtPaylink is used to pay current member registration amount
= RESPlot link is used to reserve a plot for one week for current member
= SelPlot link is used to finalize and buy a plot for the selected member
= Pymts link is used to pay emi, part payment and view your payments for current member

Projects - Used to Search,Manage project actions, create, modify , delete, create plots, delete plots, view reserved plots and combinations
= Workflow - Add Project -> Add plots -> Update Plots with imp information > buyers choose plots.
= Search the projects using name, city, desc, lead and choose the action on the right side to perform.
= Edit link is used to edit the project details , except projectid
= Del link is used to delete the project from the database
= Add link is used to add new plots to the selected project
= View link is used to view the plots available for the current project

Inquiry - Used to Search,Manage buyer queries, create, modify , delete, send quotations, add dates to call back and more combinations..
= Search the enquiry using proj, name, and choose the action on the right side to perform.
= Front office who are calling potential buyers can also add their name, to track daily targets
= Edit link is used to edit the enquiry details.
= Del link is used to delete the enquiry from the database

Reports - Used to Search,view various kinds of member, project, enquiry and plot reports..
= select the appropriate reports for viewing payments, reserving plot, viewing the plots

Gallery - Used to View project images, videos, approval records and other project updates.
